#moc #organization

## README
1. Create a MOC file with a `UniqueMocName`
2. Any notes that you want to appear in the MOC should contain a link back to the MOC
	1. e.g. `[[UniqueMocName]]`
3. Any note linked back to the MOC will show up in the index.
4. Once you add an actual link to the MOC, it'll remove automatically remove from the inbox. Now you know to sort your inboxed notes.
