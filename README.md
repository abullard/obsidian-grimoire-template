# 📖 obsidian-grimoire-template
A template Obsidian Vault to categorize documentation–utilizing organic growth via a hierarchical [Map of Contents](https://obsidian.rocks/maps-of-content-effortless-organization-for-notes/) (MoC).

## 🏎️ Getting Started
1. Install the [Obsidian App](https://obsidian.md/)
2. Open this Vault in Obsidian
    - It'll ask you to trust & enable extensions
3. Install the [Things](https://publish.obsidian.md/tasks/Reference/Status+Collections/Things+Theme) theme via Obsidian's built in theme manager
    - You can skip this, just expect to see strange checkbox syntax that doesn't render properly. e.g. `- [I]`, `- [!]`, etc.

✅ That's it, you should be up and running.

## ✏️ Contributing to YOUR Grimoire
1. Follow the Getting Started steps to install the template Grimoire
2. Checkout the [Introduction](./Diátaxis/Grimoire/MoC%20-%20Grimoire.md) (inside Obsidian) to learn how various plugins build the engine of this vault
3. Study it as a reference, or start making changes to suit your note taking preference

Good luck, have fun!
