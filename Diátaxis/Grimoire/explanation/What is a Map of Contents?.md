### [[MoC - Grimoire|← Parent]]

📘 A *Map of Contents (MoC)* is a landing page for an area of study. It contains *backlinks* to other pages, organized by category. Most often these link to other MoCs–once you hit the narrowest topic, it becomes a Table of Contents for documents related to that topic. 

> [!note]
> A backlink has the syntax `[[Backlink|Display Text]]`.
>
> With this in mind, this working example [[MoC - Grimoire|Display Text]], will take you back to the MoC for the Grimorie's tutorial.

This composition results in a drill-in hierarchy from the widest topics to the narrowest topics. For example: 

| Wide Topic   | Narrow Topic | Narrower Topic |
| ------------ | ------------ | -------------- |
| Horticulture | Tomatoes     | Caged Setup    |
| Horticulture | Tomatoes     | Staked Setup   |
| React        | Hooks        | useEffect()    |
| React        | Hooks        | useMemo()      |
| and so on…   |              |                |


> [!example] Try it out!
> Let's get a feel for how MoCs allow us to navigate without opening a single folder in the file explorer. Before you move on, try to navigate from the [Table of Contents](Table%20of%20Contents) back to this page, without using the Introduction link.
