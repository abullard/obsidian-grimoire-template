## Declare your Templates Folder

> [!NOTE] `Cold Storage/Templates`
> This Grimoire should already have a Templates folder location set.

1. Open settings
2. Navigate to "Templates" on the left
3. Supply "Template folder location" with your directory

## Insert A Template
### Use the Command Palette 
1. `cmd+p` to open Obsidian's Command Palette
2. Search for "Templates: Insert Template"
![[command_palette_insert_template.png]]
3. Search for the template you want to insert and confirm
### Use a Keyboard Shortcut
1. Review your keyboard shortcuts–mine is `cmd+alt+t` 
2. This brings up the insert template dialog
3. Search for the template you want to insert and confirm:
![[search_for_template.png]]
